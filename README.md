# Smart Home App - Concept Mobile Design

A mobile UI concept for a Smart Home application, designed to simplify how users control and automate their home devices through a single iOS app.  
Built as a personal project to explore clean UI structure, intuitive navigation, and seamless IoT integration experience within the Apple ecosystem.

---

## 1. Overview
This project focuses on creating a smooth and user-friendly interface for controlling connected home devices.  
Users can easily manage smart appliances such as lamps, curtains, and other IoT devices — all connected under one centralized app.

---

## 2. Key Features
- Onboarding screens introducing setup and connectivity  
- Home dashboard with device overview and quick actions  
- Login and guest access options for first-time users  
- Device cards with status control (on/off, mode, etc.)  
- Quick action buttons for automation (e.g., Sleep, Arrive, Get Up)  
- Bottom navigation for fast access (Home, Devices, Settings)

---

## 3. Design Focus
- UI Style: Minimalist, friendly, and visually soft for home environments  
- UX Goals: Simple navigation, quick device control, and clarity of information  
- Component System: Card-based layout with modular scalability  
- User Journey: Onboard → Connect devices → Control → Automate routines

---

## 4. Tools and Technologies
- Frontend : Swift & SwiftUI  
- Backend : Vapor  
- UI & UX : Figma  
- Database : PostgreSQL  
- IDE : Xcode  

---

## 5. About This Project
This is a conceptual and educational project, created as part of exploring iOS design and development patterns for IoT and smart environments.  
It aims to demonstrate:
- UI/UX design flow for IoT-focused applications  
- Integration planning between mobile UI and IoT backends  
- The potential of combining SwiftUI and Vapor for smart automation systems
